# racing-priofits-daily-naps-scraper

Requirements
    Node
    Chromium

Installation
    Dependencies
        `$ npm install`

    install scrapers as CLI
           `$ npm install -g`

Execution
    naps: `$ naps`
    forecasts `$ forecasts`
    all : `$ selections`

Output
    Scraped data will be in csv files in the following files
        '/Users/{user}/Documents/Selections/naps.csv'
        '/Users/{user}/Documents/Selections/forecasts.csv'
